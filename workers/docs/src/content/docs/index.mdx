## Quick Install

```bash

# or
pnpm add @cf-agents/google @cf-agents/discord
```

## Available Packages

| Package | Description |
|---------|-------------|
| [`@cf-agents/google`](/packages/google) | Gmail, Calendar, and Sheets integration with OAuth 2.0 |
| [`@cf-agents/discord`](/packages/discord) | Send messages and read channel history |
| [`@cf-agents/slack`](/packages/slack) | Post messages and read conversations |
| [`@cf-agents/telegram`](/packages/telegram) | Send messages to Telegram chats |
